This project requires:
	Houdini version 19.5.640
	Unreal Engine version 5.1.1
	Houdini Engine plugin for Unreal

The Houdini and Unreal version you have to install yourself.

The Houdini Engine plugin is included in this project, below are the instructions on how to install it.



Copy and past the folder named "HoudiniEngine" into your Unreal Engine plugins folder.

You can find this folder in your Unreal Engine install location, by default this is "C:\Program Files\Epic Games\UE_5.1\Engine\Plugins"



Note the plugin version included ONLY works for Houdini version 19.5.640 and Unreal Engine version 5.1.1 !!
